## 📥 Download

Go to the **[Releases](https://github.com/NexDam/nexdam-releases/releases)** page and download the latest version for your platform.

### Android
1. Download the `.apk` file
2. On your device go to **Settings → Security → Unknown sources** and enable it
3. Open the downloaded `.apk` and install

### Windows
1. Download the `.exe` installer
2. Run the installer and follow the setup wizard
3. Launch NexDam from the Start menu or desktop shortcut

### Linux
1. Download the `.AppImage` file
2. Make it executable:
```bash
chmod +x NexDam-*.AppImage
```
3. Run it:
```bash
./NexDam-*.AppImage
```

## 📋 What is NexDam?

NexDam is the official mobile and desktop app for [nexdam.it](https://www.nexdam.it). It allows clients to:

- 📊 Track project status in real time
- 💬 Exchange messages with the NexDam team
- 📁 Download deliverables and files
- 🧾 Check invoices and payments
